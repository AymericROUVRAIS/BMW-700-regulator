# BMW 700 regulator
This is a modern replacement of B6-08-0060 and B7-18-0600

---

The PCB can be found in the gerble file. Schematics are in the pdf file, in this file there are also the original BMW 700 documents.

The electronics used are :
- Voltage regulator *LM393*
- Mosfet
- TVS diode
- Zener diode
- Freewheling diode 
- Relay
- Resistors *10k* and *100* ohms
- Capacitor *100nF*
